# developing-universal-skills

这是一个专门用于存放和维护 Codex Skill 的仓库。

当前仓库主要用于：

- 集中管理可复用的 skill
- 保存每个 skill 的定义文件与相关资源

## 当前 Skill

### `frontend-contract-style`

用于约束前端接口契约修改时的实现方式，强调统一字段、统一调用方式，以及清理ai易生成的兼容接口内容。

### `frontend-design`

用于生成具有明确视觉方向和较高设计完成度的前端页面、组件或界面实现。
